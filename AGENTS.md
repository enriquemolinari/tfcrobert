# Descripcion

- Este es un proyect React para crear una presentacion con slides para presentar para un trabajo final de carrera de la carrera de Licenciatura en Informatica de la Unversidad Nacional de Río Negro.

- El titulo del trabajo es: Implementación de Infraestructura Open Source para el proceso de Integración Continua y Despliegue Continuo en el Desarrollo de Software.

- El trabajo esta escrito en texto_principal.md y contiene imagenes marcadas en el texto como IMAGEN diagramas/[nombre_del_diagrama].graphml que estan ubicadas en la carpeta que se denomina diagramas. Son archivos con extension .graphml hechos con el software yEd.

- Despues el resto es todo el proyecto React que quiero que utilices como modelo para crear uno igual con el texto que se encuentra en texto_principal.md.

# Restricciones

En ningun caso utilices el caracter: –
Podes utilizar este: -

# Slides

- Que sean slides con poco texto, el orador se encargara de explayarse.
- Sobre las imagenes, pone lo mas importante que esta descripto en texto_principal.md
